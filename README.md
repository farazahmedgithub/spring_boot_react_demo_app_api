# React Demo BE

## Prerequisites
- JDK 1.7 or later
- Maven 3 or later
- MySQL with DB as per application.properties

## Stack
- Spring Security
- Spring Boot
- Spring Data JPA
- Maven

## Build
``` mvn clean install```
## Run
```mvn clean spring-boot:run```
